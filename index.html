¿Qué vas a aprender?

Fundamentos de
programación

Variables, tipos de datos, condicionales, bucles y funciones. La base de todo.

HTML & CSS

Creá páginas web desde cero. Aprendé a estructurar contenido y a darle estilo con diseño responsive

JavaScript

Dale vida a tus páginas. Eventos, manipulación del DOM, formularios y lógica del lado del cliente

Backend básico

Introducción a servidores, APIs y bases de datos. Entendé cómo funciona la web por dentro

Proyecto final

Construís una aplicación web completa, funcional y lista para mostrar en tu portfolio.
